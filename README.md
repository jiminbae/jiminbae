### 🚀 Key Projects


**다변량 시계열 예측을 위한 채널 유사성 기반 스파이크 인코딩 기법 연구**
- 시계열 데이터 내 다중 채널 간 관계를 유사성을 기반으로 모델링
  - Spiking Neural Network 기반 시계열 예측 선행 연구의 채널 관계 미활용 문제 인식
  - 어텐션 메커니즘 기반 채널 관계 모델링을 통해 다변량 시계열 내 채널 관계 정량화
  - STE 기반 채널 유사도 정보를 스파이크 형태로 변환 및 스파이크 인코딩에 결합
  - 채널 정보의 결합으로 기존 스파이크 인코딩 방법의 정보 손실 문제 해결


**AI Agent 기반 민원 부서 추천·악성 민원 필터링 시스템 개발**
- 민원 내용을 수집·분석하여 배정 담당자에게 추천 부서를 제시하는 End-to-End 파이프라인 구현
  - LLM 임베딩과 VectorDB 유사도 검색을 활용, 과거 민원 기반 부서 추천 기능 구현
  - 대구 달서구 온라인 민원 데이터 대상으로 Zero-shot 정확도 78.64% 달성

  
- 민원 담당자가 악성 민원에 직접적으로 노출되지 않도록 심리적 완충 기능 구현
  - 욕설과 희롱, 비꼼 등 부정적 내용을 사전에 감지 후 일시적 흐림 처리
  - 중복 민원 이력 파악 후 사전 차단을 통해 민원 담당자의 업무 효율성 향상
